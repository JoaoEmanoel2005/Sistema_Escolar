Boa dia, como estão ?

Neste projeto, faremos uma aplicação de gestão de uma escola no quesito administrativo geral. A primeira vista, parece ser algo superficial, mas garanto que ha bem mais por tras de tudo.
Bem, primeiro vamos falar sobre as tecnologias que serao trabalhadas neste projeto, primero faremos uso dos famosos html, css e javascript para a construção da aplicação, e tambem usaremos Node e Express para a parte de Api.

Bem, a aplicação consiste em um site de gestão escolar no qual ha uma API rodando para fazer a comunicao do web com o banco de dados, que neste projeto usaremos o SQLite3.

Primeiramente, vamos preparar nosso ambiente. Após baixar e descompactar o projeto, ou clonar o repositorio, certifique-se que no Node esteja instalado em sua maquina. Partindo disso, vamos abrir o terminar dentro da pasta onde esta o projeto e digitaremos os seguintes comanados:

npm install

npm install express

npm install sqlite3

npm install cors
